# Conversor-e-compressor-de-Imagem-e-v-deo
Aplicação web para **converter e compactar imagens (PNG/JPG → WEBP)** e **comprimir vídeos (MP4)** utilizando **Node.js, Express, Sharp e FFmpeg**.   O objetivo do projeto é oferecer uma ferramenta rápida, simples e eficiente para otimização de arquivos multimídia, reduzindo significativamente o tamanho final sem perda perceptível de qualidade.
